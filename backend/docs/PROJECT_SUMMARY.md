# 📦 项目概览

## 🎯 项目目标

将 s01-s23 的零散 AI Agent 功能整合成一个完整的**网页版 Claude**，提供：
- ✨ 简洁的聊天界面
- 📁 文件上下文管理
- 🛠️ 强大的工具调用能力
- 💾 会话持久化

## 📂 核心文件说明

### 后端 (Python)

| 文件 | 行数 | 核心功能 |
|------|------|----------|
| `backend/main.py` | ~200 | FastAPI服务器、REST API、WebSocket |
| `backend/agent_service.py` | ~400 | Agent核心逻辑、会话管理、流式对话 |
| `core.py` | ~600 | 工具定义和实现 (继承自s01-s23) |

### 前端 (React + TypeScript)

| 文件 | 行数 | 核心功能 |
|------|------|----------|
| `frontend/src/App.tsx` | ~80 | 主应用、路由逻辑 |
| `frontend/src/components/ChatInterface.tsx` | ~250 | 聊天界面、WebSocket连接 |
| `frontend/src/components/Sidebar.tsx` | ~80 | 会话列表、导航 |
| `frontend/src/components/FileBrowser.tsx` | ~120 | 文件浏览器 |
| `frontend/src/api.ts` | ~60 | API客户端封装 |

## 🔄 完整工作流程

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 用户在前端输入消息                                       │
└────────────┬────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. ChatInterface 通过 WebSocket 发送到后端                 │
└────────────┬────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. AgentService 接收并构建完整消息历史                     │
│    • 包含历史对话                                            │
│    • 包含上下文文件内容                                      │
└────────────┬────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 流式调用 Anthropic API (Claude)                         │
│    • 发送消息 + 工具定义                                     │
│    • 实时返回文本 token                                      │
└────────────┬────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. 处理响应                                                  │
│    ├─ 纯文本？ → 流式返回前端显示                           │
│    └─ 工具调用？ → 执行工具 → 返回结果 → 继续循环           │
└────────────┬────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────┐
│ 6. 保存完整对话到磁盘 (.sessions/<id>.json)                │
└─────────────────────────────────────────────────────────────┘
```

## 🚀 快速开始命令

```bash

cd ai-agent-project

# 2. 配置环境
cd backend
cp .env.example .env
# 编辑 .env 添加 ANTHROPIC_API_KEY

# 3. 一键启动
cd ..
./start.sh  # Linux/Mac
start.bat   # Windows

# 4. 访问
# http://localhost:3000
```

## 🔧 可用工具

系统集成了以下工具（继承自 s02, s14）：

1. **bash** - 执行Shell命令
   ```python
   # 示例: "列出当前目录文件"
   # Claude会调用: bash(command="ls -la")
   ```

2. **read** - 读取文件
   ```python
   # 示例: "读取README.md"
   # Claude会调用: read(path="README.md")
   ```

3. **write** - 写入文件
   ```python
   # 示例: "创建hello.py文件"
   # Claude会调用: write(path="hello.py", content="...")
   ```

4. **grep** - 搜索文件内容
   ```python
   # 示例: "在所有py文件中搜索'Agent'"
   # Claude会调用: grep(pattern="Agent", path=".", recursive=True)
   ```

5. **glob** - 文件路径匹配
   ```python
   # 示例: "找到所有Python文件"
   # Claude会调用: glob(pattern="**/*.py")
   ```

6. **revert** - 回滚文件修改
   ```python
   # 示例: "撤销对config.py的修改"
   # Claude会调用: revert(path="config.py")
   ```

## 🔐 安全考虑

### ⚠️ 当前实现（开发环境）
- 无用户认证
- 工具无权限限制
- 文件系统完全访问
- 命令执行无沙箱

### ✅ 生产环境建议
1. **添加认证**
   - JWT Token
   - OAuth2 / Auth0
   
2. **权限控制**
   - 集成 s15 权限系统
   - YAML 配置白名单
   
3. **沙箱隔离**
   - Docker 容器
   - 虚拟环境
   - chroot jail
   
4. **审计日志**
   - 记录所有工具调用
   - 记录文件访问
   - 异常告警

## 📈 性能指标

### 响应时间
- **首Token延迟 (TTFT)**: ~300-500ms
- **流式更新频率**: ~50ms/token
- **工具执行**: 取决于具体操作

### 资源占用
- **后端内存**: ~100-200MB (空闲)
- **前端内存**: ~50-100MB
- **存储**: ~1MB/100条消息

### 并发能力
- **WebSocket连接**: 无限制
- **推荐最大**: 100并发用户 (单实例)
- **扩展**: 使用 Gunicorn workers
